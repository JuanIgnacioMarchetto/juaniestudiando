Conceptos HTML, CSS y JavaScript

  • Destructuring: Descomponer objetos o arreglos en variables individuales.
  • Circuito corto: Comportamiento de operadores lógicos.
  • Try, Catch, Finally: Manejo de excepciones en JavaScript.
  • Async/Await: Gestionar funciones asincrónicas.
  • Objeto literal: Estructura de datos clave-valor.
  • Fetch (peticiones): Realizar solicitudes HTTP.
  • Local Storage: Almacenamiento de datos en el lado del cliente.
